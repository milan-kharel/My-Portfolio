
+# My Portfolio Starter
+
+A clean, single-page portfolio template built with **HTML + CSS + JavaScript**.
+
+## Sections included
+
+- Home / Hero
+- About
+- Skills
+- Projects
+- Education & Achievements
+- Contact
+
+## How to run
+
+Open `index.html` directly in your browser, or run a local server:
+
+```bash
+python3 -m http.server 4173
+```
+
+Then visit `http://localhost:4173`.
+
+## Customize
+
+- Replace placeholder text in `index.html` with your real details.
+- Update colors and spacing in `styles.css`.
+- Add your real links and resume URL in the contact section.
